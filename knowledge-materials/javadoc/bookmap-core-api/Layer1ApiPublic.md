# Layer1ApiPublic

**Package:** velox.api.layer1.annotations

**Type:** Annotation Interface

## Description

Items marked with this annotation are exposed as part of L1 API. Note that commenting it out will still cause javadoc to be generated.

"@_Layer1ApiPublic" (without _) is used as a criteria for javadoc generation.
