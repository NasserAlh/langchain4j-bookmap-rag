# Layer1SimpleAttachable

**Package:** velox.api.layer1.annotations

**Type:** Annotation Interface

```java
@Retention(RUNTIME)
public @interface Layer1SimpleAttachable
```

## Description

Marks classes that can be loaded from GUI as a strategy. Simplified version of Layer1 API, for more advanced (but more complex) version use [`Layer1Attachable`](Layer1Attachable.html).
